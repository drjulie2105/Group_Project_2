Project Summary

Our group created an interactive data visualization of the presidential election data from 2000 to 2016. We obtained the data from the following sources:
1. MIT Election Lab Data
2. Election Project

The data was obtained as CSV files and cleaned using Jupyter notebook. A database was created in Postgres for the data to be imported into. Visualizations were created utilizing JavaScript, D3, HTML and CSS. Then a flask server was created to run the HTML, CSS and JavaScript files.

CSV data files were converted into geoJson files for mapping data. Mapbox was used to create the base layer map.
